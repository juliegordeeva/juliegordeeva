## Проверки по чек-листу ТЗ

| Проверка | Статус | Комментарий |
|----------|--------|-------------|
| Горизонтальный скролл | OK | scrollWidth = innerWidth на всех ширинах |
| Тач-таргеты ≥44×44 | OK | `.btn` 56px+, `.nav-toggle` 44×44, пункты меню ≥44px высоты |
| Кегль ≥16px | OK | на мобильном body = 16px |
| Переносы H1 | OK после правки | новый H1 короче по смыслу; clamp на мобильном |
| Мобильное меню | OK | все пункты доступны; закрывается после клика по якорю |
| Hero-фото | OK | на мобильном под CTA, не перекрывает текст |
| Sticky-шапка | OK | `scroll-padding-top` учитывает высоту шапки |
| Cookie-баннер | N/A | на сайте нет баннера и нет подключённой Метрики; формулировка в privacy-policy смягчена |
| Форма / чекбоксы | OK | чекбоксы 22×22 + крупная зона label; fallback-текст под кнопкой |
| Якорные ссылки | OK | меню закрывается и скролл идёт к секции |
